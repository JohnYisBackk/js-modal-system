# Modal System PRO ✨

## 🌐 Live Demo

👉 https://johnyisbackk.github.io/js-modal-system/

---

Modern dark UI **modal (popup) system** with dynamic content, smooth animations, and interactive feedback.

Built using pure HTML, CSS, and vanilla JavaScript.

---

## 🚀 Features

* 📦 Reusable modal component
* 🎯 Dynamic content (based on clicked button)
* 🧠 Action-based logic (delete / save / upgrade)
* ⌨️ Keyboard support (ESC to close)
* 🖱️ Overlay click to close
* 🔁 Focus management (returns to last clicked button)
* 🚫 Scroll lock when modal is open
* 🔔 Toast notifications with progress bar

---

## 🎮 How It Works

1. Click any action button (Save / Delete / Upgrade)
2. Modal opens with dynamic title and description
3. Click **Confirm** to trigger action
4. Toast notification appears with progress bar
5. Modal closes and focus returns to previous button
